🔍 Hướng dẫn sử dụng Web tìm kiếm Ca dao - Tục ngữ

📋 Tổng quan

Đây là ứng dụng web tìm kiếm ca dao, tục ngữ Việt Nam sử dụng thuật toán RankNet để xếp hạng kết quả theo độ liên quan.

🚀 Cách chạy ứng dụng

Phương pháp 1: Chạy tự động (Khuyến nghị)

Double-click vào file start_web.bat và chờ ứng dụng khởi động.

Phương pháp 2: Chạy thủ công

Bước 1: Mở Command Prompt trong thư mục dự án

pip install -r requirements.txt

Bước 2: Chạy ứng dụng

python app.py
Lưu ý: Nếu app.py bị lỗi, hãy thử:
python app_simple.py

Bước 3: Truy cập ứng dụng

Mở trình duyệt và truy cập: http://localhost:5000

✨ Tính năng

Giao diện tìm kiếm đẹp mắt như Google
Xếp hạng thông minh với thuật toán RankNet
Hiển thị điểm số liên quan cho mỗi kết quả
Làm nổi bật từ khóa tìm kiếm
Gợi ý từ khóa phổ biến
Thiết kế responsive, thân thiện với mobile
Thời gian tìm kiếm nhanh chóng

🎯 Cách sử dụng

  1. Nhập từ khóa: Gõ từ khóa vào ô tìm kiếm (ví dụ: "ăn", "học", "làm")
  2. Tìm kiếm: Nhấn nút "Tìm kiếm" hoặc phím Enter
  3. Xem kết quả: Kết quả được hiển thị theo thứ tự điểm số từ cao đến thấp
  4. Thử gợi ý: Click vào các từ khóa gợi ý để tìm kiếm nhanh

📊 Giải thích điểm số

Mỗi kết quả hiển thị một điểm số thể hiện mức độ liên quan:

  • Điểm cao (> 5.0): Rất liên quan đến từ khóa
  • Điểm trung bình (1.0 - 5.0): Có liên quan
  • Điểm thấp (< 1.0): Ít liên quan

🔧 Xử lý sự cố

Lỗi: "Import flask could not be resolved"
➡️ Chạy: pip install flask torch numpy
Lỗi: "File not found"
➡️ Kiểm tra file cadao_tucngu_mini.json trong thư mục FetchData/output/
Không có kết quả tìm kiếm
➡️ Thử từ khóa khác hoặc kiểm tra dữ liệu đầu vào
Mẹo: Nếu gặp lỗi với app.py, hãy sử dụng app_simple.py để test giao diện với dữ liệu mẫu.

📁 Cấu trúc file

  • app.py - Server Flask với RankNet đầy đủ
  • app_simple.py - Server Flask đơn giản để test
  • templates/index.html - Giao diện web
  • start_web.bat - Script khởi chạy tự động
  • requirements.txt - Danh sách thư viện cần thiết
